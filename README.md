# Kymo (kymo-ios) 
**Open-Source ECG & HRV Recorder for Polar Devices**

![Platform](https://img.shields.io/badge/Platform-iOS-blue)
![License](https://img.shields.io/badge/License-MPL_2.0-green)
![Status](https://img.shields.io/badge/Status-WIP-orange)

Kymo is a free and open-source iOS app that logs all data from your Polar heart rate monitor.

This is Work in Progress. Feedback is welcome!

## Features (wip)
- [x] Time-Synced 130Hz ECG
- [x] Advanced BLE Engine (automatic reconnections, device storage, signal quality etc)
- [x] Live Data Visualisations (live HR charting, dynamic HR Zones etc)
- [x] Session Tracking: Complete sport selection & background recording
- [x] Lap Management: Trigger laps manually with lap-specific statistics
- [x] Production UI: Done, highly Responsive UI
- [ ] Session Persistence + local export (csv and fit)
- [ ] Direct sync to Intervals.icu
- [ ] HealthKit integration
- [ ] GPS tracking

## Getting Started
Kymo is not live on the App Store yet! To try it now you can build it yourself:

1. Clone the repo: `git clone https://github.com/jonkwl/kymo-ios.git`
2. Open in Xcode
3. Sign the app locally with your personal Apple Developer Team
4. Build and run on your iPhone

A guide to sideloade without a Mac will come after the first ipa is out.
